= fontist-archive-public

Public archive of Unicode coverage data and open-license WOFF font specimens for the Fontist project.

== Purpose

This is the *public consumer* of `fontist/fontist-archive-private`. It contains:

* ALL coverage data (metadata is public — even for proprietary fonts)
* Open-license WOFF specimens ONLY (OFL, Apache, MIT, etc.)

It does NOT contain:

* Raw font files (proprietary — in private repo only)
* WOFF specimens for macOS/Microsoft fonts (proprietary licenses)
* Build code (this is a consumer, not a builder)

== Structure

----
coverage/{formula_slug}.yaml          Unicode coverage (ALL fonts — public metadata)
  google/abeezee.yaml
  manual/inter.yaml
  macos/font8/akaya.yaml               (coverage IS public even for macOS)

woff/{formula_slug}/{font}.woff       WOFF specimens (open-license ONLY)
  google/abeezee/esDR31xSG.woff
  manual/inter/Inter-Regular.woff
  (NO macos/ directory — proprietary)

fonts.json                            Font registry
----

== Sync pipeline

This repo syncs from `fontist/fontist-archive-private` periodically:

1. Clone private repo
2. Copy `coverage/` (ALL files — metadata is public)
3. Copy `woff/` excluding `macos/` (proprietary specimens excluded)
4. Copy `fonts.json`
5. Commit + push

No processing happens here. This is a pure consumer.

== Data separation

[cols="1,1,1,2", options="header"]
|===
|Data |Private |Public |Reason

|Raw font files |✅ ALL |❌ NEVER |Contains proprietary fonts
|Coverage YAML |✅ ALL |✅ ALL |Metadata only, no font data
|WOFF specimens |✅ ALL |✅ Open-only |WOFF is a usable font file
|Build code |✅ |❌ |Public is consumer only
|===
